# SKI_5V600A8CH_Cycler

SK 이노베이션 대전 연구소 5V 600A 8CH Demo 장비

Demo의 목적 : 설치된 유엠테크 장비 개조를 위한 사전 검증

## Overview

  - Specification
      - Voltage
        - Range : 0 ~ 5V
        - Resolution : 1mV
        - Accuracy : FS ±0.05%
      - Current
        - Range : 0 ~ 600A
        - Resolution : 1mA
        - Accuracy : FS ±0.1%
      - Charge
        - Control Mode : CC, CV, CP, CC/CV, CR, Rest
      - Discharge
        - Control Mode : CC, CV, CP, CC/CV, CR, Rest
      - Waveform
        - Rise & Fall Time : < 10ms
        - Switching Time : < 20ms
        - Over & Under shooting : < 1%
